Kurzer Erklärartikel zu „Spread“-Operator mit Beispiel

Durch den Spread-Operator kann man Iterables wie unter anderem Listen, Arrays und Strings zerlegen und deren Elemente in anderen Datenstrukturen verteilen. Der Operator wird mit 3 Punkten erstellt und ermöglicht das Hinzufügen, Kombinieren und Erstellen von neuen Iterables. Im Folgenden werde ich dazu ein paar Beispiele darstellen:

Beispiel 1: Zusammenführen von Arrays: 

                
                const array1 = [1, 2, 3];
                const array2 = [4, 5, 6];

                const zsmGefuehrt = [...array1, ...array2];
                // ergebnis: [1, 2, 3, 4, 5, 6]

Beispiel 2: Kopieren von Arrays: 

                
                const arrayOriginal = [1, 2, 3];
                const arrayKopie = [...originalArray];
                // Ergebnis: [1, 2, 3]

Beispiel 3: Spread Operator in Funktionen:

                
                function adddieren(x, y, z) {
                    return x + y + z;
                  }
                  
                  const werte = [1, 2, 3];
                  const sum = addieren(...werte);
                  // Ergebnis: 6
                  //hier werden die werte als argumente an die funktion übergeben

Beispiel 4: Neues Array mit Elementen aus einem anderem Array

                
                const alt= [1, 2, 3];
                const neu = [...alt, 4, 5];
                // Ergebnis: [1, 2, 3, 4, 5]

Wie man sieht kann man mit dem Spread Operator sehr viel machen, wie zB Iterables manipulieren, Datenstrukturen in verschiedenen Kontexten erstellen, erweitern und kombinieren.

Hier sind ein paar Beispiele die ich gebaut hab als ich Arrow Functions gelernt habe: 

                
                <script>
                // ohne parameter
                const hi = () => {
                    console.log("hiii");
                };
                

                // mit einem parameter
                const quadrat = (x) => {
                    return x * x;
                };
                

                // ein parameter + impliziter rückgabewert
                const mal3 = (zahl) => zahl * 3;
        

                // mit einem parameter
                const durch2 = (x) => {
                    return x / 2;
                };
        

                // mehrere parameter
                const addieren = (a, b) => {
                    return a + b;
                };
        

                // als funktion in einem objekt
                const hund = {
                    laut: "wuff",
                    machLaut: function(){
                        console.log(`${this.laut}`);
                    }
                };
        

                hi();
                console.log("5^2= ", quadrat(5));
                console.log("3*3= ", mal3(3));
                console.log("8/2= ", durch2(8));
                console.log("3+2= ", addieren(2, 3));
                hund.machLaut();
        

                // als call back funktion
                const zahlen = [1, 2, 3, 4, 5];
                const alleHoch2 = zahlen.map((zahl) => zahl * zahl);
                console.log("1, 2, 3, ,4, 5 quadriert: ", alleHoch2);
        

                // mit bedingung
                const istGerade = (zahl) => zahl % 2 === 0;
                console.log("4 gerade?", istGerade(4));
        
                const werte = [1, 2, 3, 4, 5];
        

                // map() mit arrow function
                const quadrierteZahlen = werte.map((zahl) => zahl * zahl);
                // forEach() arrow function
                quadrierteZahlen.forEach((zahl, index) => {
                    console.log(`Die Quadrierung von ${zahlen[index]} ergibt ${zahl}`);
                });
                // forEach() mit anonymer Funktion
                werte.forEach(function(element) {
                    console.log(element);
                });
             </script>
